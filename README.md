# Проект: Путешествие по России

Вёрстка страницы с адаптацией под десктопные и мобильные браузеры. Ширина: 1280px, 1024px, 768px, 320px.

**Используемые технологии:**
 
* Flex-box
* Grid
* Медиазапросы
* Псевдоэлементы
